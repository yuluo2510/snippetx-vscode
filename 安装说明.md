# SnippetX 安装说明 (无VSIX方式)

## 🔧 当前状态解释
**为什么已安装扩展列表看不到？**
- 你是在 **开发调试模式** 下运行
- 扩展仅在调试窗口可用，不会出现在主列表

## ✅ 立即解决问题

**方案A - 直接安装：**
1. 在VS Code中：`Ctrl+Shift+P` → "Extensions: Install from VSIX..."
2. 项目路径：`E:\Projects\newprojects` (当前文件夹即可作为VSIX)

**方案B - 手动打包：**
1. 在项目根目录新建`.vscodeignore`:
   ```
   .git/
   node_modules/
   src/
   *.log
   ```

2. 如果看到扩展，或需要立即安装：
   - 重装项目为完整扩展
   - 或使用现有文件作为dev模式
